# Asimov — Strategic Ops Intern Application
 
A single-page application site built to apply for the Strategic Operations Intern role at [Asimov (YC W26)](https://tryasimov.ai).
 
## What's inside
 
Four ops insights derived from reading the JD line by line, a why me section mapped to specific past work, and an 8-week plan with concrete deliverables — all in one deployable `index.html`.
